# Todo App Flutter

A simple and elegant todo list app built with Flutter.

## Cmd

```bash
# build

# run app

#------------
# debug cmd
#------------
```

## Ref

- https://ithelp.ithome.com.tw/users/20119550/ironman/2221

## Features

- Add new tasks with a text field
- Check/uncheck tasks to mark them complete
- Delete tasks with a delete button
- Clean, minimal UI with Material Design 3

## Project Structure

- `lib/main.dart` - Main app with todo list functionality
- `pubspec.yaml` - Flutter project configuration
- Basic project directories (lib, android, ios, test)

## Getting Started

### Prerequisites

Install Flutter by following the official guide: https://flutter.dev/docs/get-started/install

### Installation

1. Install dependencies:
```bash
flutter pub get
```

2. Run the app on a connected device or emulator:
```bash
flutter run
```

## Usage

The app provides a simple interface to manage your tasks:
- Type a task in the text field and press the + button or Enter to add it
- Tap the checkbox to mark a task as complete/incomplete
- Tap the delete icon to remove a task
